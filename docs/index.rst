Python jira_svc
###########
Python library to work with jira_svc APIs

This documents the ``jira_svc`` python package (version |release|), a Python library designed to ease the use of the
jira_svc REST API. Some basic support for the jira_svc Agile / jira_svc Software REST API also exists.

Documentation is also available in
`Dash
<dash-feed://https%3A%2F%2Fjira_svc.readthedocs.io%2Fen%2Flatest%2Fjira_svc.xml>`_ format.

The source is stored at https://github.com/pycontribs/jira_svc.

The release history and notes and can be found at https://github.com/pycontribs/jira_svc/releases
